🌐 DNS & Networking Basics

🎯 Objective
Understand how DNS works and how websites are connected through IP addresses and networks.

---

🧠 What I Learned

- DNS (Domain Name System) converts domain names into IP addresses  
- Every website is hosted on a server with a unique IP address  
- Browsers use DNS before loading any website  
- IP addresses are used to identify devices on a network  
- Networking is the foundation of all web communication  

---

🔍 Key Concepts

- Domain Name → Human-readable website name (example: google.com)  
- IP Address → Numerical address of a server (example: 142.250.x.x)  
- DNS → System that maps domain names to IP addresses  
- Server → Machine that stores and serves website data  

---

🛠 Commands Practiced

- Show current directory
    pwd
- DNS lookup
    nslookup google.com
- Test network connectivity
    ping google.com
  
 --- 
 
🧪 Practical Understanding

When a website is opened:
Browser sends request
DNS converts domain → IP address
Browser connects to server using IP
Server sends response back

 ---

💡 Key Takeaways

DNS is essential for accessing websites
IP addresses are the real location of websites
Networking commands help understand how data moves
This knowledge is important for bug bounty recon phase
